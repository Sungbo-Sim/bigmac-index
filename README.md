# 🍔 Big Mac Index — Interactive World Map

> A love letter to McDonald's, built to make global economics feel real.

**🌐 Live Demo**: [bigmac-index.vercel.app](https://bigmac-index.vercel.app)

---

## About

The **Big Mac Index** is a purchasing power parity indicator published by The Economist since 1986.

This project makes that data personal — enter your hourly wage and instantly see how far it goes around the world, measured in Big Macs.

### Features
- 🗺️ World heatmap — see which countries your wage goes furthest
- 🍔 Emoji visualization — feel the difference with burgers, not just numbers
- 💰 Personal calculator — input your wage for instant global comparison

### Tech Stack
`HTML` `CSS` `JavaScript` `D3.js` `Chart.js`

### Run Locally
```bash
git clone https://github.com/Sungbo-Sim/bigmac-index.git
cd bigmac-index
open index.html
```

---

## Why I built this

I love McDonald's. 🍔🍟🥤

I was surprised that a single Big Mac costs so differently across countries. It started with a simple question: *"How painful would it be to buy a Big Mac in Switzerland on my wage?"*

Seeing data is one thing. Feeling it is another.

## 한국어

**빅맥 지수**는 이코노미스트지가 1986년부터 발표해온 각국 구매력 평가 지표입니다.

내 시급을 입력하면 전 세계에서 빅맥을 몇 개 살 수 있는지 바로 비교할 수 있습니다.

저는 맥도날드를 정말 좋아합니다. 빅맥 가격이 나라마다 다르다는 게 신기해서, 데이터를 눈으로 보는 것보다 직접 느낄 수 있는 웹사이트를 만들었습니다.

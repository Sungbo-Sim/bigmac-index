# 🍔 빅맥 지수 인터랙티브 지도 | Big Mac Index Interactive Map

> 맥도날드를 사랑하는 사람이 만든, 세계 물가를 빅맥으로 느끼는 웹사이트

**🌐 라이브 데모**: [bigmac-index 바로가기](https://나의아이디.github.io/bigmac-index)

---

## 🇰🇷 소개

빅맥 지수(Big Mac Index)는 이코노미스트지가 1986년부터 발표해온
각국 구매력 평가 지표입니다.

이 프로젝트는 그 데이터를 **인터랙티브하게** 체험할 수 있도록 만들었습니다.

### ✨ 주요 기능
- 🗺️ **세계 지도 히트맵** — 내 시급으로 각 나라에서 빅맥을 몇 개 살 수 있는지 한눈에
- 🍔 **이모지 구매력 시각화** — 숫자가 아닌 빅맥 🍔🍔🍔으로 체감
- 💰 **나의 빅맥 구매력 계산기** — 시급을 입력하면 바로 비교

### 🛠 기술 스택
- Next.js / React
- Tailwind CSS
- D3.js

## 🇺🇸 About

The **Big Mac Index** is a purchasing power parity indicator published
by The Economist since 1986.

This project makes that data **interactive and personal** —
enter your hourly wage and instantly see how your purchasing power
compares across the world, measured in Big Macs. 🍔

### ✨ Features
- 🗺️ **World Heatmap** — See which countries your wage goes furthest
- 🍔 **Emoji Visualization** — Feel the difference with burger emojis, not just numbers
- 💰 **Personal Calculator** — Input your wage for instant global comparison

### 🛠 Tech Stack
- Next.js / React
- Tailwind CSS
- D3.js

## 🍟 만든 이유

저는 맥도날드를 정말 좋아합니다. 🍔🍟🥤

빅맥 하나의 가격이 나라마다 이렇게 다르다는 게 신기했고,
"내 시급으로 스위스에서 빅맥을 사면 얼마나 아픈 거지?" 라는
단순한 궁금증에서 시작했습니다.

데이터를 보는 것보다, **직접 느끼는 것**이 중요하다고 생각했습니다.
